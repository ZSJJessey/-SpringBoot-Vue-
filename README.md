# -SpringBoot-Vue-
• 独立完成医院管理系统的需求分析、数据库设计、系统开发与测试，覆盖挂号、排班、病历、药品、检查及统计分析等核心业务流程。 • 采用 SpringBoot + Vue 前后端分离架构实现多角色业务系统，并结合 JWT 与权限控制机制保障管理员、医生、患者等不同角色的访问边界。 • 在原有业务系统基础上探索 AI 增强能力，设计病历问答、病历摘要生成、就诊信息检索等功能原型，提升医疗文本处理与信息查询效率。 • 结合规则模板、Prompt 调优与 RAG 检索思路，设计医疗文本结构化处理方案，并为系统后续接入智能问答能力预留扩展接口。 • 结合 Redis 缓存、模块化接口设计与联调测试，优化系统在多用户访问场景下的响应效率，并为后续智能化功能接入预留技术空间。
